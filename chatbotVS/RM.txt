Руководство пользователя для Telegram бота "Заметки"

Добро пожаловать в руководство пользователя для Telegram бота "Заметки". Этот бот поможет вам создавать, просматривать и управлять заметками с использованием базы данных MySQL.

1. Начало работы

1.1 Запуск бота
   - Откройте приложение Telegram.
   - Найдите бота по его имени или ссылке, предоставленной администратором.
   - Нажмите кнопку "Start" или введите команду /start.

1.2 Главное меню
   - После запуска бота вы увидите главное меню с тремя кнопками:
     - Создать заметку
     - Посмотреть заметки
     - Удалить заметку

2. Создание заметки

2.1 Создание новой заметки
   - Нажмите кнопку "Создать заметку".
   - Бот попросит вас ввести имя заметки. Введите имя заметки и отправьте сообщение.

2.2 Добавление пунктов в заметку
   - Введите описание пункта и отправьте сообщение.
   - После этого бот попросит вас ввести количество баллов за этот пункт. Введите число и отправьте сообщение.
   - После ввода этого пунтка вы можете добавить еще один пункт или завершить создание заметки.

2.3 Завершение создания заметки
   - Вы можете повторять процесс добавления пунктов, пока не закончите создание заметки.
   - Когда закончите, нажмите кнопку "Назад", чтобы вернуться в главное меню.

3. Просмотр заметок

3.1 Просмотр списка заметок
   - Нажмите кнопку "Посмотреть заметки".
   - Бот покажет список ваших заметок.

3.2 Просмотр пунктов в заметке
   - Нажмите на название заметки, чтобы посмотреть пункты внутри.
   - Бот покажет описание каждого пункта, количество баллов за него и статус (выполнено или не выполнено).

4. Управление пунктами заметок

4.1 Отметить пункт как выполненный
   - В списке пунктов нажмите на пункт, чтобы отметить его как выполненный.
   - Бот обновит статус пункта и покажет актуальное состояние.

5. Удаление заметки

5.1 Удаление заметки
   - Нажмите кнопку "Удалить заметку".
   - Бот покажет список ваших заметок.
   - Нажмите на название заметки, которую хотите удалить.
   - Бот удалит заметку и подтвердит удаление сообщением.

6. Команды и их использование

- /start - запуск бота и отображение приветственного сообщения.
- /help - отображение руководства пользователя.
